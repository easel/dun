# Glossary

- **Agent loop**: Repeated execution of an AI coding agent with deterministic feedback.
- **Check**: A unit of validation executed by Dun.
- **Gate**: A required or optional condition for Helix workflow progression.
- **Prompt envelope**: Structured prompt-as-data payload emitted by Dun.
- **Plugin**: A manifest-defined bundle of checks and rules.
- **Phase**: Helix SDLC stage (frame, design, test, build, deploy, iterate).
